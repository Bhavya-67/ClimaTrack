📌 Project Overview
1.ClimaTrack is a real-time weather intelligence web application built using Python and Streamlit.
2.The platform allows users to monitor weather conditions, compare cities, visualize climate trends, and explore global weather insights.
3.This application uses the OpenWeather API to fetch live weather data and presents it through an interactive and user-friendly dashboard.

✨ Features
🌤 Real-Time Weather Monitoring
Get current weather conditions for any city worldwide.

🌍 Global Weather Map
Visual map representation of weather locations.

📊 Weather Forecast
View forecast data for upcoming days.

🗺 Compare Cities
Compare temperature and weather conditions between two cities.

📈 Climate Trends Visualization
Temperature trend graphs using Matplotlib.

🚨 Weather Alerts
Displays weather alerts and warnings.

🚀 Installation
1️⃣ Clone the repository
git clone https://github.com/yourusername/climatrack.git

2️⃣ Navigate to the project folder
cd climatrack

3️⃣ Install dependencies
pip install -r requirements.txt

4️⃣ Run the application
streamlit run app.py

🔑 API Configuration
Get your API key from:
👉 https://openweathermap.org/api⁠�

Python
